# The Nest Refugee Home — Website

A warm, welcoming website for **The Nest Refugee Home**, a ministry of Owen Sound Alliance Church that provides transitional housing and community support for refugee claimants in Owen Sound, Ontario.

## Pages / Sections

- **Hero** — Mission tagline with call-to-action buttons
- **About** — Who we are and what we do
- **Mission** — Four pillars: Housing, Friendship, Practical Support, Prayer
- **Get Involved** — Donate, Volunteer, Pray + current volunteer needs
- **Contact** — Contact info and message form
- **Footer** — Links and church details

## Files

| File | Purpose |

| `index.html` | Main single-page website |
| `styles.css` | All styles (responsive, mobile-first) |
